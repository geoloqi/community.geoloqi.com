    Please refer to Apple's Location Awareness Programming Guide, specifically the section on adding "location" to the list of required background modes. 

    Basically you need to add the key "location" to your UIBackgroundModes array in the app's plist file. That should allow the app to use location updates in the background.

    <mode> location </ mode>
